# SakuraQA レビューツール

4択クイズをレビューし、結果をAWS S3に保存するWebアプリケーションです。

## 使い方

### 1. アクセス

**本番環境（GitHub Pages）:**
https://kuramitsulab.github.io/SakuraQAReview/

### 2. レビュー手順

1. **レビュアー名**を入力
2. **カテゴリ**を選択
3. 問題に**回答**
4. 必要に応じて**コメント**を入力
5. **次の問題へ**進む
6. 完了後、**統計**が表示される

### 3. データの保存先

- ブラウザの`localStorage`に即座に保存
- データはAWS S3 (`sakuraqa-review-results/review.json`) に自動保存される
- ホーム画面の**「📊 分析を見る」**ボタンからデータをダウンロードできるので、AWSについては基本スルーで大丈夫です。

### 4. 分析機能

ホーム画面の**「📊 分析を見る」**ボタンから結果を分析できます。

**確認できる情報:**
- **全体正答率** - すべてのレビューの統計
- **レビュアー別正答率** - 各レビュアーの成績（グラフ・表）
- **問題作成者別正答率** - AIモデル別の正答率比較
- **問題別難易度** - 各問題の正答率（正答率順）

**使用可能な機能:**

1. **レビュアーのフィルター設定**
   - テスト用や練習用のレビュアーを統計から除外できます
   - 使い方：「レビュアー選択」セクションでチェックボックスを操作
   - 「すべて選択」「すべて解除」ボタンで一括操作も可能
   - フィルター変更後、統計が自動的に再計算されます

2. **データのダウンロード**
   - **全データ (JSONL)**: すべてのレビュー結果を詳細にダウンロード。AWS S3に保存されるのと同じ状態のものをダウンロード可能。
   - **問題別集計CSV(推奨)**: 各問題の統計をExcelで開ける形式でダウンロード。表計算ソフトで簡単に分析できます。分析するならこちらをダウンロードしてください。

```json
  "keyword": "キーワード",
  "category": "カテゴリ名",

  "choice": ["選択肢1", "選択肢2", "選択肢3", "選択肢4"],
  "answer": "正解となる選択肢",
  "year": "いつの時代の問題か",
  "reference_url": "参考になるURL",
  "authored_by": "GPT, Gemini, Claude, human...."
}
```
